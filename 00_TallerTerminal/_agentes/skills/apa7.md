# Citas APA 7

Genera entradas bibliográficas correctas para el proyecto de investigación urbana.

## Plantillas frecuentes

**INEGI:**
```
Instituto Nacional de Estadística y Geografía.
(2020). [Título]. INEGI.
https://www.inegi.org.mx/programas/ccpv/2020/
```

**CONEVAL:**
```
Consejo Nacional de Evaluación de la Política
de Desarrollo Social. (2020). [Título]. CONEVAL.
https://www.coneval.org.mx
```

**PDU Municipal:**
```
[Municipio], Gobierno Municipal. (año).
Plan Municipal de Desarrollo Urbano [municipio].
[Editorial]. [URL]
```

**Artículo académico:**
```
Apellido, N. (año). Título del artículo.
Nombre de la Revista, volumen(número), páginas.
https://doi.org/xxxxx
```

## Reglas

- Consulta web: siempre agregar `(Consultado en [mes año])`
- Sin URL disponible: agregar `(s.f.)`
- Organismo como autor: nombre completo primero
- Al generar la entrada → guardar en `/01_Investigacion/Referencias/referencias.md`
